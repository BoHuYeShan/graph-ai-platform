# 投诉与下架 / Takedown

## 如何提交投诉

如果你认为 B.A.I.T. 上的某些内容涉及侵权、误认、骚扰或对你造成困扰，
请通过以下任一方式联系我们：

- **GitHub Issues**：[提交 Takedown 请求](https://github.com/bohuyeshan/bait-core/issues/new?template=takedown.yml)
- 通过 GitHub Issue 提交时请包含以下信息：
  - 你的姓名或代表方
  - 涉及内容的 URL（DIO 编号或页面链接）
  - 投诉原因
  - 你的联系方式

## 处理时限

- 我们将在 **24–72 小时内** 对投诉做出临时处理
- 处理期间，被投诉内容将被暂时下架
- 维护者会审后做出最终决定：删除、改写、保留或加注
- 所有处理记录写入 `takedown_log.json`

## 快速下架原则

以下情况先下架再讨论：
- 真实人物本人或代理人提出异议
- 内容被用于站外骚扰或冒充
- 页面被截图误用为真实发表证明
- 内容涉及危险实验或违法风险

## 外部冒充

如果在站外发现有人声称代表 B.A.I.T. 联系你或他人，
请截图并通过上述渠道报告。此类行为违反社区公约，
不代表 B.A.I.T. 社区。
