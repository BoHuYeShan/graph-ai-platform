# 社区行为准则 / Code of Conduct

## 核心原则

B.A.I.T. 是一个以讽刺学术出版系统为目标的虚构创作社区，不是针对具体个人的攻击工具。

## 硬规则

### 禁止行为
- 主动联系真实人物本人或其亲友
- 去抖音、快手、B站、知乎、贴吧等评论区留言「你被发表了」
- 私信告知「我们帮你整理成论文了」
- 伪造「某人窃取你的成果拿奖/发表」的话术
- 组织网友围观、嘲讽、举报、钓鱼或截图传播
- 在站内对真实个人进行人身攻击、侮辱或隐私挖掘
- 使用 B.A.I.T. 名义对外沟通

### 要求
- 对事不对人
- 讽刺目标是系统和现象，不是具体的人
- 创作和讨论中保持对他人尊严的尊重
- 发现违规行为时通过 Abuse Response 渠道报告

## 违规处理
1. 警告
2. 移除相关文稿或评论
3. 暂停社区权限
4. 永久移除

## 举报
请通过 GitHub Issues（选择 Abuse Report 模板）或仓库中的 [ABUSE_RESPONSE.md](./ABUSE_RESPONSE.md) 渠道举报。
