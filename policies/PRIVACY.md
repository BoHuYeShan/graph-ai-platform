# 隐私说明 / Privacy

## 数据最小化

B.A.I.T. 是纯静态站点。我们不运行后端服务器、不收集用户行为数据、不做用户画像。

## 投稿表单（将来）

如果将来部署投稿表单：
- 只收集笔名、可选邮箱、投稿内容
- 邮箱仅用于发送录用/拒稿通知，发送后从服务端删除
- 不收集 IP、设备信息、真实身份
- 提供数据导出与删除入口

## 第三方服务

- 静态站点部署在 GitHub Pages（受 GitHub 隐私政策约束）
- 评论系统（如将来使用 Giscus）受 GitHub 隐私政策约束
- 数学渲染使用 MathJax CDN（jsDelivr）

## Cookie

本站不使用 Cookie。

## 联系方式

隐私相关问题请通过 GitHub Issues 联系。
